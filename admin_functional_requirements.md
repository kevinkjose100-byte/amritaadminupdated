# Amrita Books Admin Portal - Functional Requirements & User Flows

This document details all functional requirements and admin user flows across the ten core modules of the Amrita Books Admin Portal.

---

## 1. Dashboard
The Dashboard serves as the central operational hub, providing real-time metrics, revenue trends, weekly orders, and high-priority system alerts.

### Administrative User Flows:
* **View Key Metrics**: 
  - Track **Active Subscribers** count (with month-over-month percentage change and "Live" status badge).
  - Track **Gross Revenue** for the current month (with month-over-month percentage change).
  - Monitor **Pending Orders** count (physical orders awaiting shipment, with percentage change).
* **Analyze Revenue Trend**:
  - View the interactive area chart displaying monthly gross revenue over a rolling 6-month period (e.g. October to March).
  - Hover over data points to inspect exact revenue details.
* **Monitor Weekly Orders**:
  - View the bar chart displaying order volumes day-by-day for the current week (Monday through Sunday).
* **Manage System Alerts**:
  - View high-priority alerts generated by the system (e.g., "Low Inventory" for specific language variants, "Failed Delivery" RTS alerts).
  - Click **View** on an alert card to jump to the corresponding module (Catalog or Order Management).
  - Click **Resolve** to immediately clear the alert from the dashboard.

## 2. Catalog Management
The Catalog Management module manages the book catalog, supporting multi-language editions, physical/digital formats, regional pricing, and an OCR eBook review workspace.

### Administrative User Flows:
* **Search, Filter, & Export**:
  - Search books by title, author, or category.
  - Filter catalog list by Region (e.g., India, Rest of World, Europe, etc.), Language, Category, Format (Digital, Physical), and Published Status.
  - Click **Export CSV** to download the currently filtered catalog dataset.
* **Execute Bulk Actions**:
  - Select multiple books using checkboxes.
  - Apply bulk actions: **Delete** or **Change Status** (Draft/Published).
  - Click **Apply Discount** to open the Bulk Discount Modal:
    - Choose discount type: **Percentage (%)** or **Fixed Amount (₹)**.
    - Input the discount value.
    - Confirm to apply a sale price override across all selected books.
* **Add / Edit Book (Book-Level Metadata)**:
  - Click **Add New Book** or click **Edit** (pencil icon) on a row.
  - Configure global book metadata in **Section 1: Book Information (Common Across All Languages)**:
    - **Primary Title**: Set the global title.
    - **Author**: Select from a searchable dropdown (or click "Add New Author" option to register a new author).
    - **Series & Volume**: Select from a searchable Series dropdown (or add a new Series) and enter the volume number (e.g. Vol 3).
    - **Related Books**: Multi-select other catalog books to link them as recommendations.
    - **Categories**: Assign hierarchical categories by checking parent folders (e.g. Scripture) which dynamically displays child subcategories (e.g. Vedas, Upanishads) for selection.
    - **Featured Book**: Toggle featured status.
  - Configure language-specific details in **Section 2: Language Variants & Files**:
    - **Add Language Variant**: Select from a list of languages to initialize a new variant card.
    - **Set Default Variant**: Select one variant as the default primary language shown to store visitors.
    - **Remove Variant**: Delete an active language variant card (retaining at least one variant).
* **Configure Language Variant Cards**:
  - Click to expand/collapse a language variant card:
    - **Cover Image**: Upload via file browser or drag-and-drop (supports JPEG/PNG/WebP, up to 5MB, 1:1.5 ratio).
    - **Localized Title & Description**: Enter translation title and description for this specific language edition.
    - **Regional Pricing Table**: Inspect prices for all active country groups. Enter custom **Price** and **Sale Price** overrides for Digital and Physical formats, or use dynamic "Copy from..." options. Click "Apply India Pricing to All Regions" to copy India pricing to all regions.
    - **Format Switches**:
      - **Digital Version**: Toggle on/off. Input Digital ISBN, Base Price (₹), and Sale Price (₹).
        - **eBook Upload & Processing**: Drag-and-drop or select an eBook file (PDF/EPUB, up to 25MB) to initiate auto-OCR and quality analysis.
        - **eBook Quality Dashboard**: View file size, page count, upload timestamp, OCR Confidence, pages reviewed, issues found, and health score (Good, Moderate, Poor).
        - **Open Review Workspace**: Click to open the OCR Review Workspace.
      - **Physical Version**: Toggle on/off. Input Physical ISBN, SKU, Base Price (₹), Sale Price (₹), Stock Count, Weight (g), Dimensions (Length, Width, Height in mm), and shipping costs (domestic India vs. international).
* **Review eBook (OCR Review Workspace)**:
  - Navigate page-by-page via the left sidebar showing page-level status icons (Clean, Warning, Critical).
  - Select View Mode: **Original PDF**, **Extracted Text** (fully editable text area), or **Side-by-Side**.
  - Review page analysis in the right sidebar:
    - View OCR Confidence percentage.
    - View detected warning/critical issues (e.g., low confidence, script failures).
    - Click **Mark as Resolved** to clear a page warning, **Re-run OCR** to re-process, or **Ignore Warning**.
  - Monitor progress bar (Reviewed pages vs. total pages) and total unresolved issues count.
  - Click **Mark In Review** to change status, or click **Publish Live** (enabled only when all critical issues are resolved).

## 3. Order Management
The Order Management module tracks physical book orders, digital book deliveries, subscriptions, and coordinates fulfillment, tracking, and refunds.

### Administrative User Flows:
* **Fulfillment Filters**:
  - Filter orders list by Tab (All, Physical, Digital, Subscriptions).
  - Filter by Type (Physical, Digital, Subscription), Order Status (Pending, Shipped, In Transit, Delivered, Completed, Active, Expired, Failed, Refunded), Payment Status (Pending, Paid, Failed, Refunded, Partially Refunded), and Date Range (From/To).
  - Search by order number, customer name, or customer email.
* **View Details**:
  - Click order number or eye icon to view customer info, order type, payment status, total amount, order date, and ordered items.
* **Assign Tracking & Ship Order (Physical Orders)**:
  - For physical orders with "Pending" status, click the **Truck** icon.
  - Select Courier Service (India Post or DHL Express) and enter the tracking number (India Post numbers must end with 'IN', DHL is 10 digits).
  - Submit to save tracking details, update the order status to **Shipped**, and send a shipment confirmation email to the customer.
* **View Shipping Progress**:
  - For orders in "Shipped", "In Transit", or "Delivered" status, click the **Truck** icon to go directly to the Tracking System page with pre-populated tracking parameters.
* **Process Refunds**:
  - **Full Refund**:
    - Click **Refund Full Amount** inside the order details.
    - Input optional reason.
    - Submit to process refund, update payment status and order status to **Refunded**, and add an entry to the refund history log.
  - **Partial Refund**:
    - Click **Apply Partial Refund** (available for physical orders with multiple items).
    - Select specific items to refund using checkboxes.
    - Adjust refund quantities (system auto-calculates total refund amount based on item unit price).
    - Input refund reason.
    - Confirm to process partial refund, update payment status to **Partially Refunded**, and log the details to the refund history.
  - **View Refund Logs**: Inspect refund history cards displaying amount, date, time, reason, and list of refunded items.

## 4. Tracking System
The Tracking System integrates with courier APIs (simulated India Post/DHL) to monitor physical consignments.

### Administrative User Flows:
* **Track Consignments**:
  - Enter tracking number or order number into the tracking search box.
  - View the active shipment status (In Transit, Delivered, Return to Sender, Pending).
* **Review Milestones**:
  - View the tracking timeline showing location milestones, timestamps, status labels, and descriptions.
  - The latest milestone is highlighted and animated with a pulse indicator.
  - Delivery failures (RTS) are flagged in red.
* **Auto-Mark Completed**:
  - Shipments marked "Delivered" by the courier service auto-mark the parent order status as "Completed".

## 5. User Management
The User Management module tracks customer accounts, purchase histories, and library sizes.

### Administrative User Flows:
* **Search & View Customers**:
  - Search customer list by name or email.
  - Click the eye icon on a row to open the customer profile details:
    - View Name, Email, and Country.
    - Inspect library size (number of books owned), orders placed count, and total spent.
* **Manage Customer Account**:
  - Click **Send Password Reset Link** to dispatch password reset instructions to the customer's email.

## 6. Subscription Management
The Subscription Management Console is an administration portal for subscription plan configurations, manual user assignments, complimentary access, and lifecycle logs.

### Administrative User Flows:
* **Filter & Search Subscribers**:
  - Search by subscriber name or email.
  - Filter subscribers by Plan Type, Status (Active, Expired, Cancelled, Complimentary, Pending), Subscription Source (Purchased, Complimentary, Admin Assigned, Promotional), Expiry Date, Country, and Pricing Group.
* **Configure Membership Plans**:
  - Click the **[ + Create New Plan ]** dashed card in the grid or the "Create Plan" button.
    - Enter Plan Name (e.g. Quarterly Plan).
    - Select Billing Duration (7, 30, 90, 180, 365 Days).
    - Select Status (Active, Draft, Archived).
    - Configure Pricing: System auto-populates auto-linked price placeholders for all active pricing groups (India, Rest of World, Europe, etc.). Input custom regional prices to override, or leave blank to auto-link.
    - Enter Description and save.
  - **Inline Plan Adjustments**:
    - Select region currency dropdown in the plans header to inspect pricing per group.
    - Hover over plan cards to click **Edit Price** (updates pricing for active group) or click **Archive/Restore** to toggle plan visibility.
* **Assign Subscription Manually**:
  - Click **Assign Subscription** at the top right:
    - **Target User**: Choose "Existing User" and select from a dropdown list, OR choose "Create New" and enter their Name, Email, and Country.
    - **Select Assignment Type**:
      - **Existing Subscription Plan**: Select a plan from the list (system auto-calculates expiry date and spent amount based on selected user's country pricing group).
      - **Complimentary Access**: Set validity by inputting number of days in the numeric input box, OR select a specific expiry date. Select source type (Complimentary, Admin Assigned, Promotional) and campaign access (Amma Event Access, Book Launch Access, Volunteer Access, Ashram Member Access).
    - Click **Assign Now** to grant access.
* **View Subscriber Profile & Logs**:
  - Click the eye icon on a row to open the subscriber details panel:
    - **User Profile**: View name, email, country, library count, and total revenue contributed.
    - **Subscription Metadata**: Inspect current plan, lifecycle status, access granted by, billing source, and validity period.
    - **3-Column Logs Panel**: Inspect Payments Log, Renewal History (lists plan changes, extensions, and reactivations), and Activity & Audit History (lists administrative notes and comments).
* **Execute Lifecycle Controls**:
  - **Extend Validity**: Click "Extend Validity". Select duration (+7, +30, +90, +180, +365 days) or custom date. If subscriber status is "Expired", the extension starts from Today. Otherwise, it appends to the existing expiry date. Logs a renewal history and activity entry.
  - **Change Active Plan**: Select a plan from the dropdown list to immediately migrate the subscriber, recalculating their expiry.
  - **Convert to Paid Plan**: For complimentary subscribers, click to convert to a paid subscription (charges monthly plan price, updates status to Active, and appends payment history).
  - **Cancel Subscription**: Cancel active subscription (access remains valid until expiry date).
  - **Revoke Access Immediately**: Terminate access immediately. Sets status to Expired and updates the expiry date to today.

## 7. Pricing Models
The Pricing Models module manages regional currencies, pricing tiers, and country assignments.

### Administrative User Flows:
* **Manage Pricing Groups**:
  - Click **Create Country Group** or click **Edit Settings** on an existing group:
    - Set Group Name (e.g. Europe).
    - Select Currency (INR, USD, EUR, GBP, SGD, AED, AUD, CAD, JPY).
    - Configure Monthly and Yearly subscription prices in the local currency.
    - Set Status (Active, Draft).
    - **Country Assignments**: Search and allocate countries. The system warns if a country is already assigned to another group.
* **Configure Fallback Region (Rest of World)**:
  - View the default catch-all "Rest of World" group. Unassigned countries automatically fallback to Rest of World pricing and USD currency.
* **Delete Pricing Group**:
  - Click the trash icon to delete custom pricing groups (assigned countries automatically revert back to Rest of World).

## 8. Inventory Management
The Inventory Management module provides read-only tracking of physical stock counts across warehouse SKUs.

### Administrative User Flows:
* **Review Operational Stock Metrics**:
  - View Total SKUs tracked, Total Stock units available, and Low Stock Alerts count.
* **Monitor Low Stock Alerts**:
  - Review the Low Stock warning box, which lists book titles, languages, SKUs, and remaining units for items below the low stock threshold.
* **Audit Inventory Ledger**:
  - View the inventory table showing SKU, Title, Language, Stock Units, Weight, and Status badge (In Stock vs. Low Stock).

---

## 9. Finance & Reporting
The Finance & Reporting module monitors gateway transactions, processing fees, net revenue, and reports export.

### Administrative User Flows:
* **Monitor Finance Metrics**:
  - View Gross Revenue, Gateway Fees (Razorpay + GST), and Net Revenue for the current month.
* **Analyze Revenue Performance**:
  - View **Revenue Trend**: Line chart comparing Gross and Net revenue over a 6-month trend.
  - View **Revenue by Type**: Pie chart showcasing the revenue share of Subscriptions, Digital Sales, and Physical Sales.
  - View **Monthly Breakdown**: Bar chart comparing Gross, Fees, and Net totals.
* **Audit Transactions**:
  - View Razorpay transaction log table showing Transaction ID, Customer, Type (Subscription, Digital, Physical), Gross Amount, Fee, Net, and Date.
* **Export Financial Data**:
  - Click **Export Reports** to select and download CSV exports for:
    - Revenue Report
    - Fees Breakdown
    - Tax Report

## 10. Consignment Management
The Consignment Management module coordinates bulk tracking number ingestion and shipping codes validation.

### Administrative User Flows:
* **Submit Manual Tracking**:
  - Enter Order Number, Courier Service (India Post or DHL), and Tracking Number to register a shipment dispatch.
* **Upload Bulk Consignments**:
  - Drag-and-drop or browse to select a CSV file containing `order_number,courier,tracking_number`.
  - On upload, the system validates format rules line-by-line.
* **Audit Entry Ledger**:
  - View recent consignment entries in a table showing Order Number, Tracking Number, and Validation Status (Valid vs. Invalid, with format error descriptions).
  - India Post tracking numbers must end in `IN`. DHL Express numbers must be 10-digit numeric codes.
